Descripción del proyecto.

Este proyecto esta diseñado para mostar la ejecucion de algunos juegos mentales cinco(5), para completar cada juego, debes aprobar tres (3) veces, pero si fallas uno o colocas mal la respuesta, el juego se termina. Igual, no te preocupes, puedes volver a intentarlo. La lista de los juegos es:

- Par o Impar = Juego donde tienes que adivinar si el numero es un par o no. Recuerda que si la respuesta es que es un numero par debes colocar "yes" de lo contrario se coloca "no".

-Calculadora = Juego de aplicaion de una calculadora basica, donde tu tienes que dar la respuesta correcta.

-Maximo Común Divisor = Juego donde tienes que pensa y decir cual es el MCD entre dos numeros.

-Progression Aritmetica = Juego donde tienes que adivinar, el numero que se encuentra oculto en una secuencia de numeros.

-Es numero Primo? = Juego donde tienes que pensar si el numero que te aprace, es un numero primo o no. Recuerda que si la respuesta es que es un numero es primo debes colocar "yes" de lo contrario se coloca "no".

Instrucciones de instalación y ejecución.

1. Clonar Repositorio = gitclone https://github.com/Bam-Manu-182/python-project-140.git

2. Instalar dependencias = escribir este codigo en la terminal "pip install -e ." Para asi, instalar las dependencias.

3. Sicronizacion (Opcional) = Escribir en la terminal este código "uv sync", luego escribir este otro código "uv build" Esto sincronizara las dependencias y los scripts

4.Ejecutar juegos =

Escribe en la terminal el siguiente código para la ejecucion e inicio de cada juego:

- Par o Impar = python -m brain_games.brain_games.brain_games_par_impar

- Calculadora = python -m brain_games.scripts.brain_calc

- Maximo Común Divisor = python -m brain_games.scripts.brain_gcd

- Progression Aritmetica = python -m brain_games.scripts.brain_progression

- Es numero primo? = python -m brain_games.scripts.brain_prime
